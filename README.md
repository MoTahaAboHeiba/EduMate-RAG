# EduMate RAG System

A Retrieval-Augmented Generation system for EduMate Flutter application.

## Setup

1. Create virtual environment: `python -m venv venv`
2. Activate it: `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Configure `.env` with your Groq API key
5. Run: `python main.py`

## Project Structure

- `assets/` - PDFs and vector database storage
- `src/` - Source code
- `tests/` - Unit tests
